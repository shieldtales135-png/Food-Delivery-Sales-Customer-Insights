# Food Delivery Sales Analysis

## Overview
A simple college-level data analysis project using Python and Pandas to analyze food delivery sales.

## Objectives
- Analyze sales
- Compare city performance
- Understand customer ratings
- Study payment methods

## Tools
- Python
- Pandas
- Matplotlib

## Run
```
pip install -r requirements.txt
python analysis.py
```
